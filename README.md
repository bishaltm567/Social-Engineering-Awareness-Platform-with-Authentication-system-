# Social-Engineering-Awareness-Platform-with-Authentication-system-
A secure web application for learning and testing knowledge on social engineering threats.
A web-based cybersecurity platform designed to educate users about social engineering attacks with a secure login system and interactive quizzes.

##  Features

- 🔐 User Registration & Login System
- 🧠 Awareness Content (Phishing, Baiting, Pretexting, Tailgating)
- ❓ Interactive Quiz System
- 📊 Score Evaluation
- 🔒 Session-Based Access Control
- 💾 Secure Password Hashing

##  Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Authentication:** Werkzeug Security
- **Data Storage:** JSON (users & questions)
